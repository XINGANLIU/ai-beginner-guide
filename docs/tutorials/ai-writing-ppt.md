# AI 写作与办公

> 🎯 目标：学会用 AI 写文章、做 PPT、处理 Excel，提升工作效率。

## 用 AI 写文章

### 写邮件
```
Prompt：请帮我写一封邮件，内容是：
- 收件人：客户张总
- 目的：告知项目延期一周
- 原因：需要额外测试确保质量
- 语气：正式但诚恳
- 字数：200字以内
```

### 写工作汇报
```
Prompt：请帮我写一份周报，包含以下内容：
- 本周完成的工作：[列出要点]
- 遇到的问题：[列出问题]
- 下周计划：[列出计划]
- 格式：分条列出，每条不超过2句话
```

### 写文章/博客
```
Prompt：你是一位科技博主，请写一篇关于"2025年最值得使用的5款AI工具"的文章
- 字数：1500字
- 风格：轻松有趣
- 结构：开头引入 + 5个工具介绍 + 总结推荐
```

### 润色改写
```
Prompt：请帮我润色以下文字，使其更加专业和流畅：
[粘贴你的文字]
```

## 用 AI 做 PPT

### 推荐工具
- **Gamma** (https://gamma.app)：AI 生成 PPT 最方便
- **ChatGPT + Canva**：先让 AI 写大纲，再用 Canva 设计
- **美图 AI PPT**：国产，支持中文

### 工作流
```
1. 用 ChatGPT 生成 PPT 大纲和内容
2. 用 Gamma / Canva 导入内容自动生成设计
3. 微调布局和配色
4. 导出
```

## 用 AI 处理 Excel

### ChatGPT Code Interpreter（最强方案）
```
1. 上传你的 Excel 文件给 ChatGPT
2. 告诉它你想做什么：
   - "帮我分析销售数据的趋势"
   - "找出异常值"
   - "生成一个对比图表"
3. AI 自动处理并返回结果
```

## 更多办公场景

### 用 AI 做数据分析
```
Prompt：我有一份销售数据表格（上传文件），请帮我：
1. 找出销售额最高的 3 个产品
2. 分析月度趋势
3. 找出异常数据
4. 生成一个对比柱状图
```

### 用 AI 写 Excel 公式
```
Prompt：我需要计算 Excel 中 A2:A100 范围内大于 80 的数字的平均值，
请给我公式并解释每个部分的含义
```

### 用 AI 翻译长文档
```
方法1：上传 PDF/Word 给 ChatGPT/Claude，让它翻译
方法2：使用 DeepL（https://deepl.com）— 翻译质量最高的在线翻译工具
方法3：使用沉浸式翻译浏览器插件 — 网页双语对照翻译
```
