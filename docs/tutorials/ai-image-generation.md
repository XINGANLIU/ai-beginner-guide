# AI 生成图片教程

> 🎯 目标：零基础学会用 AI 生成高质量图片。

## 最快的上手方式

### 方法一：用 ChatGPT（GPT Image 2）
```
1. 打开 ChatGPT
2. 直接描述你想要的图片
3. AI 自动生成
```

**Prompt 示例**：
```
画一只橘色的猫坐在窗台上，窗外是下雨的城市夜景，赛博朋克风格，霓虹灯光
```

### 方法二：用 Midjourney
```
1. 加入 Midjourney Discord 服务器
2. 在频道中输入 /imagine
3. 输入你的图片描述
4. 等待生成，选择喜欢的方案
```

### 方法三：用国产工具（最简单）
```
可灵 AI：https://klingai.kuaishou.com   ← 图片+视频生成，国产标杆
通义万相：https://tongyi.aliyun.com/wanxiang  ← 完全免费
即梦：https://dreamina.jianying.com   ← 抖音生态，使用简单
```

## Prompt 写作要点

| 要素 | 说明 | 示例 |
|------|------|------|
| 主体 | 画面的核心内容 | 一只小狗 |
| 动作 | 主体在做什么 | 在草地上奔跑 |
| 环境 | 场景描述 | 春天的公园，樱花飘落 |
| 风格 | 艺术风格 | 水彩画 / 写实 / 动漫 |
| 光线 | 光线条件 | 黄金时刻 / 逆光 / 柔和光线 |
| 画质 | 质量关键词 | 高清、精细、8K |

## 不同风格关键词

| 风格 | 关键词 |
|------|--------|
| 写实 | photorealistic, ultra detailed, 8K, DSLR |
| 水彩 | watercolor, soft colors, artistic, loose brush |
| 动漫 | anime style, vibrant colors, cel shading |
| 油画 | oil painting, textured, classical, rich colors |
| 赛博朋克 | cyberpunk, neon lights, futuristic, dark |
| 像素艺术 | pixel art, retro, 8-bit, game style |
| 3D渲染 | 3D render, octane render, cinema 4D |
| 概念艺术 | concept art, matte painting, cinematic |
| 极简 | minimalist, clean, simple, modern |
| 复古 | vintage, retro, film photography, nostalgic |
