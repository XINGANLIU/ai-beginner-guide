# ChatGPT 注册与使用指南

> 🎯 目标：手把手教你注册和使用 ChatGPT，同时介绍国内替代方案。

```
注册步骤：

```
注册步骤：

**特色功能**：

## 方案二：注册 ChatGPT

### 准备工作
- 一个海外邮箱（Gmail / Outlook）
- 能够访问 OpenAI 服务的网络环境
- 一个海外手机号（可用接码平台获取）
- 一张支持国际支付的信用卡（付费版需要）

### 注册步骤

```
第 1 步：访问 https://chat.openai.com
第 2 步：点击 "Sign up"
第 3 步：输入 Gmail / Outlook 邮箱地址
第 4 步：设置密码（至少8位，含大小写字母和数字）
第 5 步：打开邮箱，点击验证链接
第 6 步：输入手机号验证（建议使用接码平台如 sms-activate）
第 7 步：选择用途（个人/工作），完成注册
第 8 步：开始使用免费版！
```

### 订阅 ChatGPT Plus
```
第 1 步：登录后点击左下角 "Upgrade to Plus"
第 2 步：选择 Plus 计划（$20/月）
第 3 步：输入信用卡信息（Visa / Mastercard）
       或使用 Apple ID / Google Play 支付
第 4 步：确认订阅
       → 可随时在设置中取消
```

## ChatGPT 界面详细介绍

| 区域 | 功能 | 说明 |
|------|------|------|
| 左侧栏 | 历史对话记录 | 可查看、重命名、删除历史对话 |
| 顶部 | 模型选择 | GPT-5.5（默认）、GPT-4o mini（更快更省） |
| 中间输入框 | 输入你的问题/指令 | 支持文字、@提及 GPTs |
| 🎤 按钮 | 语音输入 | 可用语音对话 |
| 📎 按钮 | 上传图片/文件 | 支持图片分析、文档解读 |
| 🔍 按钮 | 联网搜索 | 搜索最新信息 |
| 左下角 | 设置 | 个人资料、数据控制、订阅管理 |

## GPTs 自定义助手

ChatGPT 有一个 GPT Store，里面有数万个自定义的 AI 助手：

```
常用 GPTs 推荐：
- Canva：帮你做设计
- Code Tutor：编程导师
- PDF Reader：PDF 文件分析
- Scholar AI：学术论文搜索
- Translate：专业翻译助手
```

**如何找到 GPTs**：
1. 在 ChatGPT 左侧点击 "Explore GPTs"
2. 搜索你需要的功能
3. 点击即可开始使用

## 第一次对话示例

```
你：你好！请用中文介绍一下你自己
AI：你好！我是 ChatGPT，由 OpenAI 开发的人工智能助手...

你：帮我写一封感谢邮件，感谢同事帮我完成了项目
AI：主题：感谢您的帮助...

你：请用通俗易懂的方式解释什么是量子计算
AI：想象一下，普通电脑就像一个图书管理员...

你：（上传一张照片）这张照片里有什么？
AI：这张照片展示了一个...
```

## 实用设置

```
设置 → 个性化：
- Memory：开启后 AI 会记住你的偏好（可选）
- Custom Instructions：预设指令，每次对话自动应用

设置 → 数据控制：
- Chat History & Training：关闭后对话不会被用于训练
- 可随时删除历史对话
```
