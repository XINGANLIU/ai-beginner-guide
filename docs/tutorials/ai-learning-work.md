# AI 辅助学习与工作

> 🎯 目标：将 AI 融入日常学习和工作，成为你的超级助手。

## AI 辅助学习

### 学习新知识
```
Prompt：我想学习 [某个主题]，请像一位耐心的老师一样，
从最基础的概念开始，用简单的比喻和例子来解释，
每次只讲一个概念，讲完问我是否理解。
```

### 论文/长文阅读
```
1. 将论文 PDF 上传给 AI
2. Prompt：请总结这篇论文的核心观点、方法论和结论，
   用非专业人士能理解的语言解释
```

### 语言学习
```
Prompt：你是我的英语老师，
请和我用英语对话，纠正我的语法错误，
每次纠正时解释为什么这样改，
从简单话题开始。
```

### 制作学习计划
```
Prompt：我想在 30 天内学会 [技能]，
我每天有 1 小时学习时间，
请制定一个详细的学习计划，
包括每天学什么、用什么资源、做什么练习。
```

## AI 辅助工作

### 面试准备
```
Prompt：我要应聘 [职位]，
请模拟面试官向我提问，
每次一个问题，等我回答后给反馈。
```

### 简历优化
```
Prompt：这是我的简历 [粘贴简历]，
请帮我优化，使其更突出我的优势，
目标岗位是 [岗位名称]。
```

### 会议/沟通
```
Prompt：以下是今天的会议录音转写文字 [粘贴]，
请帮我提取：
1. 核心决策
2. 待办事项（含负责人和截止日期）
3. 需要跟进的问题
```

## AI 工作流自动化

### 常用自动化平台
| 平台 | 说明 | 适合 |
|------|------|------|
| Make (Integromat) | 可视化工作流自动化 | 办公自动化 |

