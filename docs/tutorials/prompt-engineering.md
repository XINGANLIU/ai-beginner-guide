# Prompt 提示词工程入门

> 🎯 目标：学会写好 Prompt，让 AI 输出质量提升 10 倍。

## 什么是 Prompt？

**Prompt = 你给 AI 的指令**。同样的需求，不同的写法会得到完全不同的结果。

## Prompt 写作黄金公式

```
[角色设定] + [任务描述] + [具体要求] + [输出格式] + [示例（可选）]
```

### 实战对比

```
❌ 差：帮我写一篇文章

✅ 好：你是一位资深科技记者，请写一篇 800 字的文章，
       介绍 AI 在教育领域的应用，
       要求：通俗易懂，包含 3 个真实案例，
       输出格式：标题 + 正文 + 总结
```

## 八大 Prompt 技巧

### 1. 设定角色
```
你是一位有 10 年经验的营销专家...
你是一位耐心的小学数学老师...
你是一位米其林三星厨师...
你是一位资深HR，有15年招聘经验...
```

### 2. 明确任务
```
请帮我...
- 写一封客户感谢邮件
- 总结这篇文章的 5 个核心观点
- 把这段话翻译成英文
- 制定一个7天的日本旅行计划
- 分析这份销售数据并找出趋势
```

### 3. 给出约束
```
要求：
- 字数控制在 500 字以内
- 使用正式但友好的语气
- 不要使用专业术语
- 面向高中生读者
- 包含至少2个具体例子
```

### 4. 指定格式
```
请用以下格式输出：
## 标题
### 要点（不超过5条）
- 要点1：xxx
- 要点2：xxx
### 总结（一段话）
```

### 5. 提供示例（Few-shot）
```
请模仿以下风格写一段产品描述：

示例1：「这款耳机，轻若无物，声如天籁」
示例2：「这块手表，腕间艺术，时光伴侣」

任务：为一款智能音箱写描述
```

### 6. 分步思考（Chain of Thought）
```
请一步一步思考这个问题：
1. 首先分析问题的核心
2. 然后列出可能的解决方案
3. 对比各方案的优劣
4. 最后给出你的推荐和理由
```

### 7. 角色扮演 + 对话
```
你是面试官，我是求职者。
我要应聘的岗位是产品经理。
请一个一个问我问题，等我回答后再问下一个。
每次我回答后，请给出反馈和改进建议。
```

### 8. 自我检查
```
请回答完以下问题后，检查你的回答：
- 是否有逻辑错误？
- 是否遗漏了重要信息？
- 是否可以更简洁？
如果有问题，请修正。
```

## 常用 Prompt 模板大全

### 写作类
| 模板 | 用途 |
|------|------|
| 你是一位 [角色]，请写一篇关于 [主题] 的 [文体]，要求 [约束]，格式为 [格式] | 文章/博客 |
| 请帮我写一封邮件给 [对象]，目的是 [目的]，语气 [正式/轻松]，字数 [N] 字 | 邮件 |
| 请帮我把以下内容改写成 [风格]，保持核心意思不变：[内容] | 改写润色 |
| 请为 [产品] 写 5 条广告文案，要求简短有力，突出 [卖点] | 广告文案 |

### 学习类
| 模板 | 用途 |
|------|------|
| 请像给 [N] 岁小孩解释一样，用简单的比喻解释 [概念] | 概念理解 |
| 我想学 [技能]，请帮我制定 [N] 天的学习计划，每天 [N] 小时 | 学习计划 |
| 请出一道关于 [主题] 的选择题，等我回答后告诉我是否正确并解释 | 练习测试 |
| 请扮演 [角色]，和我练习 [语言] 对话，每次纠正我的错误 | 语言学习 |

### 工作类
| 模板 | 用途 |
|------|------|
| 以下是会议记录 [内容]，请提取：1.决策 2.待办事项(含负责人) 3.风险点 | 会议纪要 |
| 请分析这份数据 [数据]，找出趋势、异常值，并给出建议 | 数据分析 |
| 请帮我把 [语言A] 的代码转换为 [语言B]，并解释关键差异 | 代码转换 |
| 请作为面试官，针对 [岗位] 对我进行模拟面试 | 面试准备 |

### 创意类
| 模板 | 用途 |
|------|------|
| 关于 [主题]，请给出 [N] 个创意想法，要求 [独特/实用/低成本] | 头脑风暴 |
| 请以 [风格] 写一个关于 [主题] 的短故事，[N] 字以内 | 故事创作 |
| 如果 [假设条件]，会发生什么？请从多个角度分析 | 思想实验 |

## Prompt 进阶技巧

### 温度（Temperature）控制

如果工具支持调参：
```
温度 0.0-0.3 → 严谨、确定（适合：事实问答、代码、翻译）
温度 0.4-0.7 → 平衡（适合：日常对话、一般写作）
温度 0.8-1.0 → 创意、随机（适合：创意写作、头脑风暴）
```

### 负面指令（告诉 AI 不要做什么）
```
注意：
- 不要使用过于口语化的表达
- 不要超过 300 字
- 不要编造数据或引用
- 不要使用"首先"、"其次"这种套话开头
```

### 多轮对话策略
```
第1轮：让 AI 生成初稿
第2轮："请改进第2段的表达"
第3轮："请更简洁，缩减到 200 字"
第4轮："请换个角度重写开头"
```

## 常见 Prompt 错误

| 错误 | 改进 |
|------|------|
| "帮我写个东西" | 明确写什么（邮件？文章？报告？） |
| "给我一些建议" | 明确什么领域的什么建议 |
| 一次问太多问题 | 拆分成多个对话，每次一个问题 |
| 没有给出上下文 | 提供背景信息让 AI 更好理解 |
| 不指定格式 | 告诉 AI 你想要的输出格式 |
