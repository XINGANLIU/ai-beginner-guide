| 天数 | 主题 | 目标 | 预计时间 |

| Day 1 | 初识 AI | 完成第一个对话 | 30 分钟 |
| Day 2 | Prompt 入门 | 学会基本提问技巧 | 45 分钟 |
| Day 3 | Prompt 进阶 | 掌握高级技巧 | 45 分钟 |

## Day 1：初识 AI（30分钟）

### 今日目标
完成你的第一次 AI 对话！

### 学习内容

**Step 1：注册 DeepSeek（10分钟）**
```
1. 打开 https://chat.deepseek.com
2. 点击"注册"
3. 用手机号或邮箱注册
4. 完成验证
```

**Step 2：完成第一次对话（20分钟）**

尝试以下对话：

```
对话 1：基础对话
你：你好，请介绍一下你自己

对话 2：实际问题
你：帮我写一封请假邮件，因为明天要去医院体检

对话 3：创意任务
你：给我讲一个有趣的程序员笑话
```

### 今日练习
- [ ] 注册 DeepSeek 账号
- [ ] 完成至少 3 轮对话
- [ ] 尝试问一个你工作/学习中的实际问题

## Day 2：Prompt 入门（45分钟）

### 今日目标
学会写出清晰、具体的提示词

### 学习内容

**Step 1：了解 Prompt 基础（15分钟）**

好的 Prompt 要素：
```
1. 角色：告诉 AI 它是谁
2. 任务：明确你要什么
3. 背景：提供相关信息
4. 格式：指定输出格式
```

**Step 2：对比练习（30分钟）**

```
❌ 不好的 Prompt：
"帮我写个文章"

✅ 好的 Prompt：
"你是一位资深科技博主，请帮我写一篇关于AI工具的科普文章，要求：
- 目标读者：完全不懂技术的小白
- 字数：800字左右
- 结构：先解释概念，再介绍3个工具，最后总结
- 语言风格：通俗易懂，多用比喻"
```

### 今日练习
- [ ] 用"不好的Prompt"提问，记录回答
- [ ] 用"好的Prompt"提问，对比效果
- [ ] 自己写 3 个不同场景的 Prompt

## Day 3：Prompt 进阶（45分钟）

### 今日目标
掌握更多 Prompt 技巧

### 学习内容

**Step 1：学习高级技巧（20分钟）**

| 技巧 | 说明 | 示例 |
|------|------|------|
| Few-shot | 给几个例子 | "类似这样：输入A→输出B，输入C→输出?" |
| 思维链 | 让AI展示思考过程 | "请一步一步思考" |
| 角色扮演 | 扮演专家 | "你是一位有10年经验的产品经理" |
| 限定范围 | 减少幻觉 | "只基于我提供的信息回答" |

**Step 2：实战练习（25分钟）**

```
练习 1：Few-shot
你：请按照以下格式帮我写产品描述：
示例：
产品：蓝牙耳机
描述：无线自由，音质出众，8小时续航，运动必备。

现在请写：
产品：智能手表
描述：

练习 2：思维链
你：请一步一步分析，为什么DeepSeek能在2025年爆火？
```

### 今日练习
- [ ] 练习 Few-shot 技巧
- [ ] 练习思维链技巧
- [ ] 总结出 5 个你最常用的 Prompt 模板

## Day 4：AI 写作（60分钟）

### 今日目标
用 AI 完成一篇完整的文章

### 学习内容

**Step 1：生成文章大纲（15分钟）**
```
你：请帮我写一篇关于"AI如何改变学习方式"的文章大纲，包含：
- 引言（为什么这个话题重要）
- 3个主要观点
- 每个观点配1个具体案例
- 结论和建议
```

**Step 2：分段生成内容（30分钟）**
```
你：请根据大纲的第一部分，写一个吸引人的开头，150字左右
```

**Step 3：润色修改（15分钟）**
```
你：请帮我润色这段文字，让它更流畅、更有说服力：
[粘贴你的文字]
```

### 今日练习
- [ ] 用 AI 生成一篇文章大纲
- [ ] 完成一篇 500 字以上的文章
- [ ] 尝试让 AI 帮你润色

## Day 5：AI 办公（60分钟）

### 今日目标
用 AI 提升日常工作效率

### 学习内容

**场景 1：邮件撰写（15分钟）**
```
你：帮我写一封商务邮件，内容是：
- 收件人：客户张经理
- 目的：邀请他参加下周的产品演示会
- 时间：下周三下午3点
- 地点：线上会议（腾讯会议）
- 语气：专业但友好
```

**场景 2：PPT 大纲（15分钟）**
```
你：我要做一个10分钟的演讲，主题是"2026年AI趋势"，
请帮我生成PPT大纲，包含：
- 标题页
- 3-4个主要内容页
- 结论页
每页列出2-3个要点
```

**场景 3：Excel 问题（15分钟）**
```
你：我有一个Excel表格，A列是日期，B列是销售额。
请帮我写一个公式，计算2026年1月的销售总额。
```

**场景 4：会议纪要（15分钟）**
```
你：请根据以下会议记录，整理一份简洁的会议纪要：
[粘贴会议记录]
要求包含：参会人、讨论要点、决议事项、待办事项
```

### 今日练习
- [ ] 用 AI 写一封邮件
- [ ] 用 AI 生成 PPT 大纲
- [ ] 用 AI 解决一个实际工作问题

## Day 6：AI 图片（45分钟）

### 今日目标
学会用 AI 生成第一张图片

### 学习内容

**Step 1：用 ChatGPT 生成图片（20分钟）**
```
1. 打开 ChatGPT（需Plus或使用免费额度）
2. 输入：
   画一只可爱的橘猫，坐在书桌前用电脑编程，
   卡通风格，温馨的灯光，窗户外面是星空
3. 等待生成，下载保存
```

**Step 2：用国产工具（25分钟）**

推荐：通义万相（完全免费）
```
1. 打开 https://tongyi.aliyun.com/wanxiang
2. 注册/登录
3. 输入图片描述
4. 选择风格
5. 生成并下载
```

### 今日练习
- [ ] 用 ChatGPT 或通义万相生成 3 张图片
- [ ] 尝试不同的描述风格
- [ ] 保存你的作品
