# 隐私与数据安全

> 🎯 目标：了解使用 AI 时的隐私风险，学会保护自己。

## 你的数据去了哪里？

当你向 AI 发送消息时：
- 内容会被发送到 AI 公司的服务器
- 可能用于改进模型（免费版通常默认同意）
- 可能存储一定时间

## 绝对不要告诉 AI 的信息

| 信息类型 | 风险 |
|---------|------|
| 密码、验证码 | 账号被盗 |
| 身份证号、银行卡号 | 身份盗用 |
| 公司内部文件 | 商业机密泄露 |
| 个人隐私照片 | 隐私泄露 |
| 未发表的学术论文 | 知识产权风险 |
| 病人/客户的个人信息 | 违反保密义务 |

## 安全使用 AI 的原则

```
1. 默认你的对话可能被存储和审查
2. 不要在对话中输入敏感个人信息
3. 处理机密文件时使用企业版（有数据保护协议）
4. 定期清理不需要的对话记录
5. 了解所用 AI 工具的隐私政策
```

## 各工具的隐私政策对比

| 工具 | 免费用户数据是否用于训练 | 付费用户 |
|------|------------------------|---------|
| ChatGPT | 是（可关闭） | 否 |
| Claude | 否 | 否 |
| DeepSeek | 查看最新政策 | 查看最新政策 |

## 各工具的隐私设置教程

### ChatGPT
```
设置 → 数据控制 → 关闭 “Chat History & Training”
→ 你的对话不会被用于训练模型
→ 可以随时删除历史对话
```

### Claude
```
Claude 默认不用用户对话训练模型（所有用户）
可以在设置中查看和删除对话历史
```

## 企业级 AI 使用建议

```
1. 使用企业版 AI 服务（有数据保护协议）
2. 制定内部 AI 使用规范
3. 明确哪些数据可以输入 AI，哪些不可以
4. 定期审计 AI 使用情况
5. 考虑部署私有化 AI 模型（如 DeepSeek 开源版本）
```
