# AI Agent（智能体）

> 🎯 目标：了解 AI Agent 的概念，这是 AI 的下一个重要方向。

## 什么是 AI Agent？

**AI Agent（AI 智能体）** 是能自主思考、规划并执行任务的 AI 系统。

```
普通 AI：你问一个问题 → 它给一个回答
AI Agent：你给一个目标 → 它自己拆解步骤 → 调用工具 → 完成任务
```

## AI Agent 和普通 AI 的区别

| 对比项 | 普通 AI（如 ChatGPT） | AI Agent |
|--------|----------------------|----------|
| 交互方式 | 一问一答 | 给出目标，自主执行 |
| 能力 | 生成文本 | 使用工具、上网搜索、操作软件 |
| 记忆 | 仅限当前对话 | 可以记住之前的操作 |
| 规划 | 被动回答 | 主动制定执行计划 |

## 生活中的 AI Agent 例子

```
"帮我订一张明天去北京的机票"
→ Agent 自动：查航班 → 比价 → 选最优 → 填写信息 → 完成预订
```

## Agent 工作流程图解

```
传统 AI：
用户输入 → AI 处理 → 输出回答

AI Agent：
用户输入目标
  → 第1步：分析目标，拆解任务
  → 第2步：调用工具 A（如搜索）
  → 第3步：根据结果调整计划
  → 第4步：调用工具 B（如计算）
  → 第5步：整合所有结果
  → 输出完整答案
```

## 用 Coze 搭建自己的 Agent（零基础教程）

```
第 1 步：访问 https://www.coze.cn，注册账号
第 2 步：点击“创建 Bot”
第 3 步：设定 Bot 的角色和指令（Prompt）
第 4 步：添加插件（如搜索、知识库、代码执行）
第 5 步：测试对话效果
第 6 步：发布到微信/飞书/网页等渠道
```

### Coze Bot 案例
```
客服 Bot：设定角色为客服 + 添加知识库（公司FAQ）
翻译 Bot：设定角色为翻译 + 添加多语言插件
简历 Bot：设定角色为HR + 添加模板工具
```
