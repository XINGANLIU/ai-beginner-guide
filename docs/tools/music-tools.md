## Suno 详细使用教程

**官网**：https://suno.com

### 注册
```
1. 访问 https://suno.com
2. 用 Google/Microsoft/Discord 账号登录
3. 免费版每天可生成 5 首（10 个积分）
4. Pro 计划 $10/月（500首），Premier $30/月（2000首）
```

### 生成音乐
```
简单模式：
1. 在输入框描述你想要的歌曲
2. 如：一首欢快的流行歌曲，关于夏天的旅行
3. 点击 Create
4. 等待 30 秒，获得 2 个版本
5. 试听、下载 MP3 或视频

自定义模式（推荐）：
1. 开启 "Custom" 模式
2. Lyrics：输入你的歌词（或让 AI 自动生成）
3. Style of Music：描述风格
4. Title：歌曲标题
5. 点击 Create
```

## Udio 详细评测

**官网**：https://www.udio.com

- **优势**：音质比 Suno 更好，音乐性强，风格控制更精确
- **价格**：免费版每月有限额度，Standard $10/月
- **特色功能**：可以延长歌曲、混音、remix
- **适合**：对音质有更高要求的用户
