## 各工具详细评测

### ChatGPT（OpenAI）

**官网**：https://chat.openai.com

**优点**：
- 全球最知名、用户最多的 AI 工具
- 插件生态丰富（GPTs 商店有数万个自定义助手）
- 支持图片理解、语音对话、代码执行、联网搜索
- 持续更新，性能稳定
- 支持文件上传和分析

**缺点**：
- 需要海外手机号注册（有难度）
- 需要科学上网
- 免费版有用量限制（高峰期可能无法使用 GPT-5.5 Pro）
- Plus 订阅价格较高（$20/月 ≈ ¥145/月）

**适合**：有条件访问的用户，追求综合能力最强

**注册教程**：
```
1. 准备：海外手机号 + 科学上网环境
2. 访问 https://chat.openai.com，点击 Sign up
3. 输入邮箱（推荐 Gmail/Outlook）
4. 设置密码（至少8位，含字母和数字）
5. 完成邮箱验证（点击邮件中的链接）
6. 输入手机号验证（可用接码平台）
7. 注册完成！开始使用免费版
```

**订阅 Plus**：
```
1. 登录后点击左下角 "Upgrade to Plus"
2. 需要一张支持外币支付的信用卡（Visa/Mastercard）
3. 或使用 Apple/Google 账号支付
4. $20/月，随时可取消
```

### Claude（Anthropic）

**官网**：https://claude.ai

**优点**：
- 擅长长文本（可处理整本书、长代码）
- 推理能力强，回答更加谨慎和准确
- 幻觉率较低
- 支持上传文档分析（PDF、Word、图片等）
- 代码能力出色

**缺点**：
- 同样需要科学上网
- 免费版对话次数有限（约 20-30 次/天）
- 不支持中文官网直接注册

**适合**：需要处理长文档、学术写作、代码开发的用户

## API 使用简介（进阶用户）

如果你想在程序中调用 AI，可以使用 API：

```python
# DeepSeek API 示例（Python）
from openai import OpenAI

client = OpenAI(
    api_key="your-api-key",
    base_url="https://api.deepseek.com"
)

response = client.chat.completions.create(
    model="deepseek-chat",
    messages=[{"role": "user", "content": "你好！"}]
)
print(response.choices[0].message.content)
```

> API 是按 Token 计费的，通常比订阅便宜很多。适合有编程基础的用户。
