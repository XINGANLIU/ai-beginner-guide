## Gamma：AI 做 PPT 详解

**官网**：https://gamma.app

### 使用流程
```
1. 注册 Gamma 账号（免费 10 个积分，可做 10 份 PPT）
2. 选择 "Generate" → "Presentation"
3. 输入主题或粘贴大纲
4. 选择风格模板
5. AI 自动生成完整的 PPT（含设计、排版、配图）
6. 微调内容、调整布局
7. 导出为 PPTX 或 PDF
```

### Gamma 使用技巧
```
✅ 先让 ChatGPT 生成详细大纲
✅ 大纲越详细，PPT 质量越高
✅ 每页控制在 3-5 个要点
✅ 导出后可以再用 PowerPoint 微调
```
