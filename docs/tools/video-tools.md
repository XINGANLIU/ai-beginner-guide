## 视频生成基本流程

```
1. 准备素材
   ├── 文字描述（Prompt）
   ├── 或参考图片（图生视频）
   └── 或参考视频（视频风格化）

2. 选择工具，输入 Prompt

3. 设置参数
   ├── 时长（通常 5-10 秒）
   ├── 分辨率（720p / 1080p）
   ├── 宽高比（16:9 / 9:16 / 1:1）
   └── 风格（写实/动画/电影感）

4. 生成 → 预览 → 调整 → 下载
```

## 视频 Prompt 写作指南

### 公式
```
[主体] + [动作] + [场景环境] + [镜头语言] + [风格] + [氛围/光线]
```

### 示例

```
电影感：
A woman walking on a beach at sunset, 
slow motion, cinematic, golden hour lighting, 
wide angle shot, shallow depth of field, 
film grain, anamorphic lens flare

产品广告：
A luxury watch rotating on a black surface, 
studio lighting, close-up macro shot, 
reflective surface, premium feel, 
dark background with subtle blue accents

自然风光：
Aerial drone shot over misty mountains at dawn, 
slow camera movement, cinematic color grading, 
4K, peaceful atmosphere, nature documentary style

动漫风：
Anime girl flying through clouds on a dragon, 
dynamic camera movement, vibrant colors, 
epic scene, Studio Ghibli inspired
```

### 常用镜头语言关键词

| 术语 | 英文 | 效果 |
|------|------|------|
| 广角镜头 | wide angle shot | 展示大场景 |
| 特写 | close-up shot | 突出细节 |
| 慢动作 | slow motion | 强调动作 |
| 航拍 | aerial shot / drone shot | 俯瞰视角 |
| 跟踪镜头 | tracking shot | 跟随主体移动 |
| 延时 | time-lapse | 加速时间流逝 |
| 变焦 | zoom in/out | 推近/拉远 |
| 平移 | pan left/right | 水平移动 |
| 倾斜 | tilt up/down | 垂直移动 |
