## 如何选择适合自己的 AI 工具？

```
你的需求是什么？
├── 聊天问答、写作、学习 → 对话类（首选）
├── 生成图片、设计素材   → 绘画类
├── 生成视频、短视频     → 视频类
├── 生成音乐、歌曲       → 音乐类
├── 提升办公效率         → 办公类
├── 写代码、做网站       → 编程辅助
└── 自动化复杂任务       → AI Agent
```

## 新手入门推荐路径

```
第 1 步：选一个对话工具（推荐 DeepSeek），开始聊天
第 2 步：学会写好 Prompt（参考 Prompt 工程教程）
第 3 步：尝试用 AI 写作/办公
第 4 步：尝试 AI 绘画/视频
第 5 步：根据需求深入某个方向
```

> 💡 **新手建议**：先从对话类工具开始，这是最容易上手、也最实用的类别。
